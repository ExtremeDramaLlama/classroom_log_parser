Missed Pay Detector

Introduction

This is a tool intended for tutors contracted with Tutor.com. The software we use, Classroom, has some flaws. One of these flaws is that it will, sometimes, falsely detect that the student has left the session. The visible symptom of this is that you recieve a popup say that the student left, even though the student is still there and the session is proceding normally otherwise.

At this point, you are no longer being paid for your time, and you will have to submit a support ticket, along with the log file, to get paid for that missing time. It's very easy to miss these incidents, which is why I created this tool that analyzes the log file and tries to detect them.
